Django App development following [PythonPro](https://www.python.pro.br/) Course

#### Install the dependencies using Pipenv

```bash
pipenv sync -d
```
#### Activate Virtualenv

```bash
source .venv/bin/activate
```
Alternatively:
```bash
pipenv shell
```
